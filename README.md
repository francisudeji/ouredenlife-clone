# frontend

## Project setup

```
yarn install
```

### Environment setup

Create a `.env` file and add the following:

```
VUE_APP_BASE_URL=https://localhost:1337
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
